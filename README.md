# witcher3-booster

Various tweaks for Witcher 3

* Enables debug console (Ingame toggle: `~`)

### Console functions
* `addmoney(11)`
* `removemoney(11)`
* `additem('gwint_card_eredin_gold')`
  *  item names in `patch0\bundles\gameplay\items`
*  `eatapple()`
*  `god()`
*  `god2()`
*  `god3()`
*  `learnskill('sword_s10')`
  * names in `patch0\bundles\gameplay\abilities` 
*  `testpause()`
*  `testunpause()`
*  `likeaboss()`
